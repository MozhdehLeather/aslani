# WireGuard VPN Auto Installer for iPhone This repository provides a **ready-to-use script** to set up a WireGuard VPN server on an **Ubuntu/Debian server in India** and generate a **client configuration for iPhone**. The script automates server setup, client key generation, configuration, firewall adjustments, and outputs a **QR code** for easy iPhone import.
